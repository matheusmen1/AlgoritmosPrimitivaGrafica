# Graphic Primitives (Primitivas Gráficas) using C#

This project has the aim to explore diferentes `Graphic Primitives`, like: `straight`, `circumferences`, `ellipses` and `polygons`.

**To show in the monitor the `straight`, we utilized this algorithms:**
- Straight Equation;
- DDA Algorithm (Digital Differential Analyzer);
- Bresenham;

**To show in the monitor the `circumferences`, we utilized this algorithms:**
- Circumference Equation;
- Trigonometry;
- Mid Point Algorithm;

**To show in the monitor the `ellipses`, we utilized this algorithms:**
- Mid Point Algorithm -> modified to the ellipses;

## 2D Transformations
Our project also aims to demonstrate the 2D Transformation that can by aplied to a polygon.
This 2D Transformations is classified in 5 big groups, tha are:

- Scale (S)
- Translation (T)
- Rotation (R)
- Shearing
- Reflection

## Polygons Rastering (fill)
Also was implemented the fill of a polygon, using 2 distinct algorithms, tha are:
- Flood Fill
- Scanline with AET (Available Edge Table)
## Tech Stack

**Client:** C#

**Server:** C#
